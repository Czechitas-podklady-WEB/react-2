## Styled Components


### Instalace

```shell
npm install --save styled-components
```

### V Reactu

```jsx
import styled from 'styled-components';
```

A potom:

```jsx
const Title = styled.h1`
 padding: 20px;
 background-color: #222;
 text-align: center;
 color: white;
 font-size: 1.5em;
`;
```

Pou≈æijeme jako:

```jsx

```
